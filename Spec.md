# Spec — 规约文档

## 页面流程

```
配置页 → 候课页 → 抽背页
  ↘ 学情看板页
```

## 各页面功能

### 配置页
- 选择班级（下拉框）
- 选择课题（下拉框）
- "上课"按钮 → 进入候课页
- "学情看板"按钮 → 进入看板页

### 候课页
- 显示当前课题的全部问题列表（供老师预览）
- 关注名单设置区
- "开始抽背"按钮 → 进入抽背页

### 抽背页
- 左侧：问题按钮（点击展示问题，可随机排序）
- 右侧：学生抽选（名字滚动动画，2秒后停下）
- 抽选逻辑：关注名单中的学生优先，抽完再抽其他人
- 停止后显示"通过/不通过"按钮，点击保存记录

### 学情看板页
- 顶部：班级筛选 + 导出CSV + 清空数据
- 三个标签页切换：
  - 按班级：总览卡片 + 各班通过率条形图
  - 按学生：表格（姓名、班级、被抽次数、通过次数、通过率），按通过率升序
  - 按课题：各课题通过率条形图
- 纯CSS条形图，不依赖外部库

## 版本管理规则（语义化版本 SemVer）

格式：`vX.Y.Z`（主版本.次版本.修订号）

| 位置 | 名称 | 什么时候 +1 | 举例 |
|------|------|------------|------|
| X | 主版本 | 大改：加新页面、改核心架构 | v1→v2：加了看板和记录 |
| Y | 次版本 | 新功能：加按钮、加新模式 | v2.0→v2.1：加关注名单 |
| Z | 修订号 | 修bug、调样式、删元素 | v2.0.0→v2.0.1：修引号bug |

递增规则：
- Z 递增时，Y 不变（v2.0.1 → v2.0.2）
- Y 递增时，Z 归零（v2.0.3 → v2.1.0）
- X 递增时，Y和Z 都归零（v2.1.0 → v3.0.0）
