# Real — 现实约束

## 必须约束（4项）

1. 纯 HTML 单文件，不依赖网络、不依赖任何外部库
2. 必须在 file:// 协议下正常运行（U盘直接双击打开）
3. 数据与代码分离（当前用 localStorage 实现，未来可换其他存储）
4. 界面必须简洁优雅，不堆砌元素，能不加UI就不加

## 可选约束（2项）

1. JS 中的中文双引号必须用 Unicode 转义（`\u201c` `\u201d`），避免与字符串定界符冲突
2. 全屏响应式布局，字号用 vw/vh 单位，适配86寸一体机（学生最远观看距离9米）

## 教室环境

- 显示屏：192×106cm（86寸鸿合一体机），触屏操作
- 教室：宽7.25m × 深9.8m，50人
- 网络：有网但不稳定，不依赖联网
