# 超级抽背大师

高中政治课堂抽背工具，纯HTML单文件。

- **项目路径**：`~/Projects/超级抽背大师/`
- **GitHub**：`caoyue46/recite-master`

## 项目结构（RCSW）

- `Real.md` — 现实约束（4条红线）
- `Cog.md` — 认知模型（实体与关系）
- `Spec.md` — 规约文档（功能、版本规则、版本历史）
- `超级抽背大师.html` — Work（教室版，纯抽背）
- `管理后台.html` — Work（教师版，看板+关注名单）

修改代码前请先读 Real.md 和 Spec.md。

## 开发规则

### 版本号（必须遵守！）
- 每次修改代码都要更新版本号，不管改动多小
- 三处同步更新：HTML title、Spec.md 版本历史、MEMORY.md 当前版本
- 语义化版本 vX.Y.Z：改样式/文案 → Z+1，加功能 → Y+1

### 每次改完代码的完整流程
1. 更新版本号（HTML title + Spec.md + MEMORY.md）
2. git add → git commit（message 以版本号开头）→ git push
3. 复制两个 HTML 到 U 盘：`/Volumes/U PAN/教学工具/`

## 设计原则

### 教室环境
- 86寸鸿合一体机（192×106cm），触屏操作
- 学生观看距离：最近2m，最远9m
- 全屏布局，响应式字号（vw/vh），不用固定像素

### 保护学生
- 不通过时不显示任何负面符号（不打叉、不标红）
- "不通过"按钮文字用"加油"，颜色用灰色（温和低调）
- 通过时显示绿色 ✓，正向鼓励

### 界面风格
- 简洁克制，双色系：靛蓝 #4f46e5 + 琥珀金 #f59e0b
- 扁平按钮，无渐变无阴影，触屏用 :active 反馈
- 能不加的元素就不加
